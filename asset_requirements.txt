ADDITIONAL NOTES:
----------------

1. All gang logos should maintain a consistent style while representing the 
   character of each gang.

2. The police, hospital, and black market icons should be easily distinguishable 
   at small sizes on the game board.

3. Sound files should be optimized for web use (compressed appropriately).

4. Font files are optional if using Google Fonts as specified in the HTML.

5. All images with transparency should use PNG format.

6. Image dimensions are recommendations - slight variations are acceptable as long 
   as the aspect ratios are preserved.
